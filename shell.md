## 使用是 shell

> Shell 是系统的用户界面，提供了用户与内核进行交互操作的一种接口（命令解释器）

Sehll可以执行：

* 内部命令

* 应用程序

* shell脚本

```mermaid

graph TB

用户输入指令-->T[提交给Shell]

T --> P{内置命令}

P --Yes--> R[内核中的系统功能调用]

P --No--> N[在系统中查找该命令的<br/>文件并调入内存执行]

N --> R

```

> 使用 type 命令可以区分内部命令和外部命令
